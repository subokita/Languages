# `적 / 적으로 / 적이다`

-   Same function as Chinese `的`
    
    -   `민주` _democracy_
    -   `민주적` _democratic_
    -   `민주적으로` _democratically_
    -   `미국은 대통령을 민주적으로 선출해요`
    -   _America elects its president democratically, LOL_
-   Adding `이다` turns the word into adjective
    
    -   `과학` _science_
    -   `과학적` _scientific_
    -   `과학젝이다` _scientific_
    -   `이것은 과학적인 문제예요`
    -   _This is a scientific problem_

# `스럽다`

-   Change noun to adjective
    
    -   `‌만족` _satisfaction_
    -   `‌만족하다` _satisfied_
    -   `‌만스럽다` _satisfactory_
    -   `저는 음식이 만족스러윘어요`
    -   _I was satisfied with the food_
-   `~게` can be added to the end of the stem `~스럽다` to change the word into adverb
    
    -   `그는 한국어를 자연스럽게 말해요`
    -   _He speaks Korean naturally_
