# `-mente` Form

-   Convert to feminine singular form, and add `-mente`
    -   `claro → clara → claramente`

# con + singular form

-   `cuidadoso` → `cuidadosamente` → `con cuidado`

# Irregular ones

-   `bastante` _quiet_
-   `demasiado` _too_
-   `mal` _badly_
-   `mucho` _a lot_
-   `muy` _very_
-   `nunca` _never_
-   `peor` _worse_
-   `poco` _little_
-   `siempre` _always_

# When two adverbs modify the same verb

-   First one uses feminine singular form, second one uses `mente` ending
    -   `El autor escribió rápida y facilmente`
    -   _The author wrote quickly and easily_
