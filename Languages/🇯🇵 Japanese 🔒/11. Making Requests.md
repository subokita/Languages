# Polite

-   Use `ください`, it’s formal version of　`くれる`
    -   `漢字で書いてください。`
    -   _Please write in Kanji_

# Casual

-   Use `ちょうだい`
    -   `ここに名前を書いてちょうだい。`
    -   _Please write your name here_

# Firm but polite

-   Attach `なさい` to verb-stem form
    -   `よく聞きなさい！`
    -   _Listen well!_
-   drop `さい` to make it more casual
    -   `それでいいと思うなら、そうしなよ`
    -   _If you think that's fine, then go ahead and do it_
