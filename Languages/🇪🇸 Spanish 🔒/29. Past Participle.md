# Formation
- Simply drop the infinitive ending and add `-xdo` endings
    - `hablar - ar + ado → hablado`
    - `comer - er + ido → comido`
    - `vivir - ir + ido → vivido`

# Irregulars

- `abrir → abierto` *opened*
- `cubrir → cubierto` *covered*
- `decir → dicho` *said*
- `escribir → escrito` *written*
- `freier → frito` *fried*
- `hacer → hecho` *done*
- `morir → muerto` *dead*
- `poner → puesto` *put*
- `resolver → resuelto` *resolved*
- `romper → roto` *broken*
- `ver → visto` *seen*
- `volver → vuelto` *returned*
