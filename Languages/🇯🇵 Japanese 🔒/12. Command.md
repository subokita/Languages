# Command

-   For `る` verb, replace `る` with `ろ`
    -   `好きにしろ！`
    -   _Do as you please!_
-   For `う` verb, change /u/ vowel to /e/ vowel
    -   `あっち行け！`
    -   _Go away!_
-   Exceptions
    -   `する → しろ`
    -   `くる → こい`
    -   `くれる → くれ`
        -   `早く酒を持ってきてくれ！`
        -   _Hurry up and bring me some alcohol!_

&emsp;&emsp;&emsp;

# Negative Command

-   Add `な` to the verb
    -   `それを食べるな！`
    -   _Don't eat that!_
