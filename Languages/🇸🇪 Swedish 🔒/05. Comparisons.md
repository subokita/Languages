# With -are, -ast

| Positive | Comparative | Superlative |
| -------- | ----------- | ----------- |
| glad     | gladare     | gladast     |
| rolig    | roligare    | roligast    |
| vacker   | vackrare    | vackrast    |
| grym     | grymmare    | grymmast    |
